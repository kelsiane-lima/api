## Instalação
É necessário ter instalado na sua máquina o [Node.js](https://nodejs.org/en/)
Em seguida:

Clona o repositório:
```sh
git clone 
```
Linux ou Windows:

```sh
npm install ou npm i
```

## Ferramentas e Tecnologias utilizadas:

* PostgresSQL
* Express.js
* Node.js
* Knex.js
